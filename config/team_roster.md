## 🔄 워크플로우

### 커밋 컨벤션
```
<type>(<scope>): <subject>

예: feat(auth): 로그인 기능 추가
```

**Type 종류:**
- `feat`: 새로운 기능
- `fix`: 버그 수정
- `docs`: 문서 수정
- `style`: 코드 포맷팅
- `refactor`: 리팩토링
- `test`: 테스트 추가
- `chore`: 빌드, 설정 변경
